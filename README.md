# Robotics and Embedded AI Portfolio

## About Me
I am an undergraduate student majoring in Electronic Science and Technology.
My main interests include robotic systems, embedded control, robotic arm development, and edge AI deployment.

## Research Interests
- Embodied Intelligence
- Robotic Control
- Embedded Systems
- Edge AI / TinyML

## Selected Projects

### 1. Robotic Arm Control for RoboMaster Engineering Robot
This project focuses on robotic arm control, system integration, motion planning, and embedded communication for the RoboMaster engineering robot.

### 2. TinyML-Based Battery SOC Estimation
This project explores lightweight deep learning model deployment on ESP32 for real-time battery state-of-charge estimation.

### 3. STM32-Based Servo Robotic Arm Control System
This project implements a robotic arm control system based on STM32, ROS, and MoveIt for motion planning and execution.

### 4. ESP32-Based LED Matrix Desktop Clock
This project is a compact embedded application based on ESP32, featuring LED matrix display control and desktop clock functions.

### 5. ESP32-Based Smartwatch Prototype
This project is a smartwatch demo based on ESP32, focusing on embedded GUI design and wearable system integration.
