## 基于 ESP32 的 HUB75 点阵屏显示系统

### 项目概述
- 基于 **PlatformIO + Arduino** 开发的 **ESP32** 点阵屏控制项目
- 面向 **HUB75 RGB LED 点阵屏** 的显示驱动与应用实现
- 结合 **I2S 并行 + DMA** 实现高速、稳定刷屏，区别于普通 GPIO 软件扫描方式

### 底层驱动与屏幕适配
- 封装点阵屏驱动类，支持画点、清屏、画线、画矩形、显示文字、显示位图、颜色转换和亮度调节
- 支持多种面板驱动芯片初始化，包括 **FM6124 / FM6126A / ICN2038S / MBI5124 / SM5266P**
- 支持多块面板拼接后的虚拟大屏坐标映射
- 支持蛇形拼接、旋转及不同扫描方式适配

### 主程序功能实现
- 按 **64×64 单块面板**、**1 块串联**完成显示配置
- 设置 **E 引脚为 32**，并指定屏驱动类型为 **FM6124**
- 初始化屏幕后设置亮度为 **80**，并完成清屏
- 开机连接 WiFi，并通过 **NTP** 获取当前网络时间
- 获取时间后进行字符串格式化，并分三行显示在点阵屏上
- 校时完成后主动断开 WiFi 并关闭无线模块，减少持续联网开销

### 显示效果与动画
- 实现彩虹渐变色 **Hello!** 文字显示效果
- 显示 **24×24** 表情位图
- 播放 **eyes_3** 图片上下切换动画，实现眼睛动态效果
- 播放 **pingu1 ~ pingu4** 连续帧动画
- 实现 **mario** 位图横向滚动显示
- 在滚动过程中绘制绿色竖线，并重复刷新时间显示

### 项目特点
- 基于 **Adafruit GFX** 扩展文字与图形显示能力
- 具备点阵屏底层驱动、动画显示、时间联网校准与应用集成能力
- 当前工程更偏向于 **ESP32 驱动 HUB75 点阵屏进行时间、文字、图片和动画展示** 的演示型应用项目
