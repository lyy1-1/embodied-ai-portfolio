## 基于 TinyML 的电池 SOC 预测

### 数据处理与模型训练
- 使用 Python 对电池数据进行预处理
- 训练 CNN、LSTM、CNN-LSTM 等深度学习模型用于 SOC 预测

### 模型量化与部署
- 使用 TensorFlow Lite 对模型进行 INT8 量化压缩
- 生成适用于嵌入式平台部署的轻量化推理模型

### 嵌入式推理实现
- 在 ESP32 平台实现模型推理程序
- 实现电池电压、电流等数据输入下的实时 SOC 预测

### 项目结果
- 完成从模型训练、模型量化到 MCU 部署与推理验证的完整流程
- 验证了轻量化 AI 模型在 MCU 上部署的可行性
